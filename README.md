# Stats Dashboard — сайт для статистического анализа

## О проекте
Это мой дипломный проект после курсов по аналитике данных.  
Сайт работает как калькулятор статистических методов.

### Какие методы реализованы:
- Описательная статистика
- Сравнение групп
- Корреляция
- Регрессия

## Что я протестировала в этом проекте
Я не просто написала сайт - я полностью протестировала его вручную:

- Тестировала граничные значения: пустые поля, нечисловые данные, выбросы
- Проверяла, правильно ли строятся графики при разных входных данных
- Находила и исправляла ошибки в формулах (в процессе отладки)
